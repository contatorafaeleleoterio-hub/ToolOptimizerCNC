Fix — Thumb Double-Translation (Tailwind v4)
Bug: thumb deslocado 50% para esquerda em todos os sliders
v0.4.3 Concluído
›

Root Cause

  • Tailwind v4 gera CSS translate (propriedade standalone)
  • style={{ transform }} gera CSS transform
  • Ambos ativos = double translation no thumb

Fix Aplicado

  • Removido -translate-x-1/2 de StyledSlider e BidirectionalSlider
  • Track: mx-[18px] em todos os 3 componentes
  • Clearance simétrico: 18px margin + 6px gap − 14px = 10px
